# 1L Navigation

一个延续 `personal-agent-site` 视觉语言的私人导航网站。

## 本地预览

这是纯静态页面，可以直接打开 `index.html`，也可以用任意静态服务预览。

## 设计方向

- 首屏使用摄影作品作为全屏封面，文字覆盖在图像之上。
- 页面保持暖白背景、近黑正文、灰褐辅助文字和细分割线。
- 标题使用 serif 字体，正文与导航使用克制的 sans-serif。
- 导航内容用目录式横排行组织，避免卡片堆叠、圆角、阴影和装饰渐变。
- 图片保留完整色彩，不做单色化处理。

## 文件结构

- `index.html`: 页面结构与导航内容
- `styles.css`: 视觉系统与响应式样式
- `app.js`: 年份等轻量交互
- `images/hero-window.jpg`: 首屏封面
